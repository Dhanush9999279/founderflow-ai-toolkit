# FounderFlow AI Toolkit
## 🌐 Live Demo

https://bold-venture-ai.lovable.app

A practical AI toolkit designed to help founders think clearly, validate ideas, and grow faster.

---

## What this is

This repository contains a structured set of tools for founders:

- High-quality prompts for startup thinking  
- A system-level AI copilot (Founder Copilot)  
- Templates to organize founder knowledge  
- Realistic examples for reference  

---

## Who this is for

- Solo founders  
- Student entrepreneurs  
- Early-stage builders  
- Anyone struggling with unclear startup thinking  

---

## How to use

1. Go to the `/prompts` folder  
2. Pick a prompt (analysis, validation, growth)  
3. Paste your idea into the prompt  
4. Use the `/system/founder-copilot.txt` for deeper thinking  
5. Store your insights using `/templates`  

---

## Why this exists

Most founders don’t fail because they lack effort —  
they fail because of unclear thinking.

This toolkit is built to:
